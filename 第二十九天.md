今日目标：debug

- `f5` -> 开始debug
- `shift + f5` -> 停止debug
- `shift + command + f5` -> 重新开始debug
- `shift + command + 9（改键）` -> 打断点
- `shift + f5` -> 停止debug
- `f10` -> 调过步骤
- `shift + command + '（默认f11与系统冲突，所以改键）` -> 进入当前步骤的函数里面
- `f5` -> 退出当前步骤

可以使用 VSspacecode插件

### 心得
我个人还是特别习惯鼠标点击去 debug 的，应为经常用 Chrome 的调试习惯了
