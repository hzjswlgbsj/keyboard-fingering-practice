进入目标：操作多个vscode窗口

- 打开新的 vscode 窗口
   - mac  `shift + command + n`
   - win  `shift + ctrl + n`
- 选择工作区 `ctrl + r`
- 多个窗口切换
   - mac  `command + ``
   - win  Easy Windown Switcher（插件）
- 关闭 vscode 窗口
   - mac  `shift + command + w`
   - win  `shift + ctrl + w`

### 总结
用 `shift + command + n` 可以直接新开一个空白的 vscode 的窗口，然后只用 `ctrl + r`打开文件选择，贼方便。

用  `command + ``切换多个项目窗口也是相当方便。

然后 `shift + command + w` 直接关闭一个 vscode 项目窗口。
