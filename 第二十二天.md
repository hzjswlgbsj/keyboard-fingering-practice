- `option + up/down` -> 将光标位置的一行代码上下移动
- `command + enter` -> 在当前行下面增加一行
- `shift + option + d` -> 将当前行的代码复制并增加到下一行
- `option + delete` -> 删除光标前面的单词（以空格为标识识别单词，实则是假单词）
- `option + ctrl + delete` -> 删除光标前面的单词（指的如果是驼峰他能一个一个的删除）
- `f8` -> 跳转到编辑器提示的错误所在位置
- `command + f2` -> 选择所有出现的当前单词
