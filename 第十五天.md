进入目标：掌握窗口管理

### 任务

- 新建窗口
   - 左右：Ctrl+w v
   - 上下：Ctrl+w s

- 窗口切换
   - Ctrl+w  hjkl
   - Ctrl+w w

- 关闭窗口
   - Ctrl+w  c
   - Ctrl+w w
- 只保留当前窗口，关闭其他所有的窗口
   - Ctrl+w  o
### 扩展
利用 vscode 的自定义快捷键

- 新建窗口
   - Command + \
   - Command + Ctrl + \
- 关闭窗口
   - Command + w
   - Command + k + w
- 窗口切换
   - shift + 方向键
   - 没有改键的话使用 shift + ctrl + jkhl
